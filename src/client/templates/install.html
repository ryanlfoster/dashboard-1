Install GitHub Dashboard

Add webhook

In your repository add the following webhook, make sure to select "send me everything". Or just select specific events, whatever, it's your life. 

{{ config.value.webhook }}

Include .dashboard.yml

In the root of your repository add a .dashboard.yml file.
If no file is present we will use default settings instead. 

logo: {{ config.value.github }}/:user/:repo/raw/master/logo.png
branch:
  - master
  - release

You're done!

Status updates and events will start coming in as they are received, head here to check out the action. 

{{ config.value.url }}/:user/:repo
