Install GitHub Dashboard

Add webhook

In your repository add the following webhook, make sure to select "send me everything". Or just select specific events, whatever, it's your life. 

{{ config.value.webhook }}

Include .dashboard.yml

In the root of your repository add a .dashboard.yml file.
If no file is present, that's cool, we will use some default settings instead. 

logo: http://...       # url to logo image
delay: 5               # minutes to display notifications
branch:                # list of watched branches
  - master
  - release
stats:                 # list of statistics
  - url: http://...
    icon: octicon octicon-star
  - url: http://...
    icon: octicon octicon-star

Statistics

Statistics are simple webservices that return a json object with a text value. This text will be displayed below the icon you provide, we poll these services every minute to keep them up-to-date.
An example payload looks like this. 

GET: http://myawesomewebservice.com/statistic

{
    "text": "Everything is awesome!"
}

You're done!

Status updates and events will start coming in as they are received, head to your reposiroty to check out the action. 

{{ config.value.url }}/:user/:repo
